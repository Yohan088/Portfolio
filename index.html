Scripts Python d'automatisation

Pack de scripts pour accélérer la préparation de données et les tâches répétitives. Les exemples ci-dessous viennent du dossier outils_automatisation ; ils sont pensés pour être utilisables par des non-développeurs (paramètres en en-tête, logs clairs).

Tri et copie sélective de dataset

Script interactif OpenCV pour parcourir des images, copier les couples image/.txt vers un dossier « keep » et suivre l'état (copié / non copié). Idéal pour filtrer rapidement un dataset annoté (YOLO).

Fichier : auto_alim_dataset

Annotation vidéo au clavier

Outil « Annotateur volley » pour extraire des frames, les labelliser (service, réception, passe, attaque, bloc) et générer un CSV + captures horodatées. Navigation rapide (flèches, J/K) et contrôle de vitesse.

Fichier : Annotateur_frame

Extraction rapide de clips

Wrapper Python au-dessus d'ffmpeg (via imageio-ffmpeg) pour découper une vidéo longue en extraits précis (durée ou intervalle), en copie sans ré-encodage ou en H.264 optimisé.

Fichier : extrait_longue_video

Nettoyage des annotations

Suppression automatique des fichiers .txt orphelins lorsqu'aucune image n'existe (sécurisé par mode simulation). Permet d'assainir un dataset YOLO avant entraînement.

Fichier : suppression_auto

Recréation locale de fonctions Roboflow

Pour garder la maîtrise des données, j'ai reconstruit en local des transformations clés façon Roboflow : flips horizontaux/verticaux par lot et crops automatiques des régions d'intérêt, intégrés à mes pipelines computer vision.

Batch image flipping

Automatisation des flips avec gestion des annotations, pour augmenter la diversité des datasets sans dépendre d'outils cloud.

Tech : Python, Pillow / OpenCV, itération dataset.

Automatic cropping

Découpage local d'images ou zones d'intérêt, intégré à mes scripts CV pour reproduire la chaîne de préprocessing Roboflow.

Tech : Python, OpenCV, automatisation filesystem.

Prototype de SaaS basé sur la vision (type YOLO) : ingestion des données, modèle CV pour détecter/segmenter, puis interface utilisateur délivrant les résultats de manière opérationnelle.
